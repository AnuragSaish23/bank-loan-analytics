# SQL Queries

All MySQL queries for data analysis.

## Contents

| File | Description |
|------|-------------|
| `01_basic_analysis.sql` | Basic counts, averages, distributions |
| `02_risk_analysis.sql` | Default rates by various factors |
| `03_financial_analysis.sql` | Income, interest rates, loan amounts |

## Database Setup

1. Create a MySQL database named `bank_loan_db`
2. Import the cleaned data from `data/processed/`
3. Run queries in order
